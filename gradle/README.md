# How to use this image

## Run a Gradle task

From your project directory:

```bash
docker run --rm -u gradle \
  -v "$PWD":/home/gradle/project \
  -w /home/gradle/project \
  riscv64/gradle gradle <task>
```

Replace `<task>` with your desired Gradle task, e.g., `build`.
